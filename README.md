# Russian-travel

### О проекте

* Учебный проект "Russian-travel". Был выполнен в ходе обучения на [Яндекс.Практикум](https://practicum.yandex.ru/). Это одностраничный сайт с информацией о интересных местах России и ресурсами про эти места.

## Стэк технологии
### HTML
- Семантические теги

### CSS
- Flexbox
- Grid Layout
- BEM Nested
- Псевдоклассы CSS
